--------

Data Expressions

When binding data, you can also use simple expressions. Hibiki HTML supports all of the standard conditional and arithmetic operators: +, -, /, *, %, ==, !=, >, <, >=, <=, !, ||, &&, ??, <=>

Hibiki also supports ternary conditionals: (condition ? true-case : false-case)

Here are some examples:

  • <div style="font-weight: *($.selected ? 'bold' : 'normal')">
  • <h-text bind="$.index + 1">

You can see live examples on the Basic Test Page. and more information is available in the Expression Reference.

Shortcuts

Instead of writing out <h-text bind="expr"></h-text> you can use {\{ expr }\} with an optional trailing format specification.

Additional classes can be turned on/off using the shorthand attribute syntax (remember to use a * to make it dynamic):
<div class.class-name="value">

Examples:

  • There are {\{$.numlights}\} light{\{$.numlights > 1 ? 's' : ''}\}
  • <button class="button" class.is-primary="*$.isprimary">
  • <div class="* 'button ' + ($.isprimary ? 'is-primary' : '')">
  • {\{ $.pi | format("%.2f") }\}
Functions

For convenience, Hibiki HTML provides a set of functions to help manipulate data in expressions. The complete list is listed in the Basic Test Page and in the Function Reference.

Functions all start with "fn:" and are not case sensitive. Here are some examples: 

  fn:max($.num, 10);
  fn:slice($.arr, 0, 5);
  fn:int($.form.strval);
  fn:split($.str, ",");
  fn:json($.form);
  fn:jsonparse('{"x": 5}');
